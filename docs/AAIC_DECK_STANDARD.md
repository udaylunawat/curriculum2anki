AAIC Flashcard Deck Standard v1.0

1. Purpose
	•	Rapid revision
	•	Concept clarity
	•	Exam + interview readiness
	•	Zero depth creep

2. Card Types (ONLY these)
	•	Definition
	•	Formula
	•	Intuition
	•	Usage context

No examples-heavy cards. No proofs. No derivations.

⸻

3. Front Rules (Immutable)
	•	≤ 12 words
	•	No punctuation
	•	Explicit prefixes allowed:
	•	Definition:
	•	Formula:
	•	Intuition:

⸻

4. Back Rules (Immutable)

Order is fixed:
	1.	Formula (if applicable)
	2.	Plain English explanation
	3.	ELI5 section mandatory

LaTeX:

[$ $] ... [/$ $]

No inline math. Ever.

⸻

5. Tagging Rules

Mandatory:

aaic::chapter_<n>::topic::prompt_<version>

Never change historical tags.

⸻

6. Volume Rules
	•	Max 18 cards per chapter
	•	One idea per card
	•	Prefer under-generation over completeness

⸻

7. Quality Bar

A card must:
	•	Stand alone
	•	Be understandable in isolation
	•	Be recallable in < 10 seconds

If it violates this, delete it.

⸻

8. Versioning Policy
	•	Prompt changes → bump version
	•	Never regenerate old cards silently
	•	Old cards are immutable once reviewed
